--- Internal ROM image

    Main SYSTEM:
	BOOT_BAS8.ROM    $FE00 - $FFFF BOOT ROM(Basic Mode)
	BOOT_DOS8.ROM    $FE00 - $FFFF BOOT ROM(Dos Mode)
	BOOT_BBL8.ROM    $FE00 - $FFFF BOOT ROM(Bubble casette Mode / optional)
	BOOT_1MB.ROM	 $FE00 - $FFFF BOOT ROM(8" Floppy Mode / optional)
	
	FBASIC10.ROM	$8000 - $FBFF F-BASIC V1.0

    Sub System:
	SUBSYS_8.ROM    $D800 - $FFFF Subsystem monitor and font.

    Optional:
	Please select one of two:
	KANJI1.ROM      JIS class 1 Kanji ROM $20000 bytes
	KANJI.ROM       JIS class 1 Kanji ROM $20000 bytes
	
    Additional info:
	* Bubble casette files are supported below:
	  a. .BBL file. This is plain binary file contains
	     32768 or 131072 bytes of any data.
	     You can make this (i.e: 32768 bytes with U*ix) :
	     	dd if=/dev/zero of=foo.bbl bs=4096 count=8
	  b. .B77 file. This has not tested because I don't
	      have any images, sorry.
