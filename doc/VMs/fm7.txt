--- Internal ROM image

    Main SYSTEM:
	BOOT_BAS.ROM    $FE00 - $FFFF BOOT ROM(Basic Mode)
	BOOT_DOS.ROM    $FE00 - $FFFF BOOT ROM(Dos Mode)
	BOOT_MMR.ROM    $FE00 - $FFFF BOOT ROM(MMR Mode / optional)
	
	Please select one of three:
	FBASIC.ROM	$8000 - $FBFF F-BASIC V3.0
	FBASIC300.ROM	$8000 - $FBFF F-BASIC V3.0
	FBASIC302.ROM	$8000 - $FBFF F-BASIC V3.0 L20

    Sub System:
	SUBSYS_C.ROM    $D800 - $FFFF Subsystem monitor and font.

    Optional:
	Please select one of two:
	KANJI1.ROM      JIS class 1 Kanji ROM $20000 bytes
	KANJI.ROM       JIS class 1 Kanji ROM $20000 bytes
	FDDSEEK.WAV     Sound file of FDD SEEKING.
	RELAY_ON.WAV    Sound file of turning on a CMT's RELAY.	
	RELAYOFF.WAV    Sound file of turning off a CMT's RELAY.
