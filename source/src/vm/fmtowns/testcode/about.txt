---------------------------------------------------------------------
 FM TOWNS 互換ROM シリーズ : FMT_SYS.ROM
---------------------------------------------------------------------

■これは何か？
--------------
FM TOWNSエミュレータうんづで使用可能な互換ROMイメージのひとつで、実機
から抽出したROMイメージではなく、独自に作成されたROMイメージです。
実機から抽出したROMイメージは、当然ながら個人利用の範囲を超えて使うこと
はできません。しかし、本アーカイブに含まれる互換ROMイメージは、実機の
もののような著作権に縛られることなく配布を行うことができます。もちろん
FM TOWNS実機を所有していない方が使っても、何ら問題はありません。

■生成方法
----------
アーカイブ内には、同梱のソースコードから生成された FMT_SYS.ROM が既に
含まれています。
ご自分で互換ROMイメージを生成する場合は、次の要領で行ってください。

1.生成に必要なファイル
互換 FMT_SYS.ROM は、次のファイル群を結合することで生成されます。結合
順序も次のとおりにしてください。添付の makesys.bat を使うと簡単に結合
することができます。

  FMT_SYS0.F12 (131072バイト) : 12ドットフォント
  FMT_SYS1.EXB ( 32768バイト) : EXT-BOOT(ダミー)
  FMT_SYS2.ICN ( 32768バイト) : システムアイコン
  FMT_SYS3.DMY ( 32768バイト) : ダミー
  FMT_SYS4.LGO ( 15360バイト) : 起動ロゴ
  FMT_SYS5.IC2 (  1024バイト) : 起動時に使うアイコン
  FMT_SYS6.PRG ( 16384バイト) : ブートコード

2.各構成ファイルの作成方法
上記のファイル郡は、それぞれ以下の手順で作成してください。

・FMT_SYS0.F12
  本アーカイブに 0xFF で埋めたダミーファイルが含まれています。この
  ファイルをお使いください。

・FMT_SYS1.EXB
  本アーカイブに 0xFF で埋めたダミーファイルが含まれています。この
  ファイルをお使いください。

・FMT_SYS2.ICN および FMT_SYS5.IC2
  BMP2ICN.EXE を使って、ICON.BMP を変換することにより作成します。
  使い方はコマンドプロンプトから、

    bmp2icn icon.bmp

  と実行してください。

・FMT_SYS3.DMY
  本アーカイブに 0xFF で埋めたダミーファイルが含まれています。この
  ファイルをお使いください。

・FMT_SYS4.LGO
  BMP2LOGO.EXE を使って、LOGO.BMP を変換することにより作成します。
  使い方はコマンドプロンプトから、

    bmp2logo logo.bmp

  と実行してください。

・FMT_SYS6.PRG
  ソースコードをNASMでアセンブルします。コマンドプロンプトから、

    nasmw -f bin sys.asm -o fmt_sys6.prg

  と実行してください。NASMの使用方法は、NASMのマニュアルをご覧くだ
  さい。

----------
・本アーカイブに含まれるプログラムは、すべてフリーソフトです。

・本アーカイブに含まれるファイルの使用から生じたいかなる損害について
  も、作者は一切責任を負いません。

・これは互換ROMであって、実機とまったく同じ動作をすることを保証する
  ものではありません。動作が実機と異なる、画面の見た目が違う等の問題
  は不具合ではありません。

・うんづを起動させるために必要なROMイメージは複数あります。本互換
  ROMファイルのみでは起動させることはできません。

・本互換ROMについて、FM TOWNSの発売元である富士通株式会社へのお問い
  合わせはご遠慮ください。


■著作権
--------
著作権は Kasanova (kasanova@retropc.net) が保有します。しかし、下記
の条件にもとづいて改変された部分に関してはこの限りではありません。


■改変・再配布条件
------------------
本アーカイブに含まれるファイルは、一部または全部を改変・再配布するこ
とができます。ただし、以下の条件を守って行ってください。

・改変をして再配布する場合は、改変を行った者と改変部分がわかるよう
  にしてください。

・生成された FMT_SYS.ROM を配布する場合は、実機から抽出されたもの
  とは異なる ROM イメージである旨を明記する文書を必ず添付してくだ
  さい。
  互換ROMイメージであることの断りなく FMT_SYS.ROM のみを配布するの
  はご遠慮ください。

なお、再配布に際してはできるだけ Kasanova までご一報くだされば幸い
です。
