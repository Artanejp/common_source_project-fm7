--- What's this ?

This archive includes the drivers for Common Source Code Project.

	yayaHX-20+		TOSHIBA HX-20 + FDD Emulator (experimental)
	yayaMSX2+		ASCII MSX2+ Emulator (experimental)
	yayaFS-A1		Panasonic FS-A1 Emulator (experimental)
	yayaPX-7		Pioneer PX-7 Emulator (experimental)
	yayaMSX1		ASCII MSX1 Emulator (experimental)
	yayaMSX2		ASCII MSX2 Emulator (experimental)

You can build them with Microsoft Visual C++ 2008 SP1 and DirectX SDK.
