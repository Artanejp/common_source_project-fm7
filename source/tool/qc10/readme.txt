How to create mfont rom

You can create mfont.rom from true type fonts.
MultiFonts ROM card supports 16 fonts and
you can specify the source fonts in mkmfont.bat:

	mkmfont.exe "source font name"

Sample mkmfont.bat refers free true type fonts.
You can download them in the website below:

http://ansuz.sooke.bc.ca/software/ocrb.php
http://www.webpagepublicity.com/free-fonts.html
http://www.free-fonts-ttf.org/true-type-fonts/
http://www.germancorner.com/fonts/index.html
